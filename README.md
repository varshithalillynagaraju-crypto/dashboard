# Dashboard UI Design

## Overview

This project is a responsive Business Dashboard UI developed using HTML, CSS, and Chart.js. It provides a clean and modern interface for monitoring key business metrics through KPI cards, charts, filters, and navigation components.

## Features

* Responsive Dashboard Layout
* 12-Column Grid System
* Sidebar Navigation Menu
* KPI (Key Performance Indicator) Cards
* Category Filters
* Date Range Selector
* Interactive Bar Chart
* Interactive Pie Chart
* Business Analytics Banner
* Business & Market-Oriented Images
* Drill-In Details Section
* Mobile-Friendly Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js

The dashboard automatically adapts to different screen sizes, providing an optimized experience on desktop, tablet, and mobile devices.
